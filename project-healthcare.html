Healthcare Analytics

NYC Healthcare Access Equity Analysis

Geospatial data analysis identifying healthcare access gaps across NYC boroughs

Python pandas NumPy scikit-learn matplotlib seaborn Jupyter

Overview

This project analyzes geospatial healthcare data across NYC boroughs using regression and clustering to identify access equity gaps in underserved communities. A segmentation framework and Tableau dashboard were built to support data-driven resource allocation for public health stakeholders.

Project Status

This project is currently in development. The full case study, visualizations, and deliverables will be published soon. Check back for the complete analysis.

Planned Approach

Business Objective

Analyze healthcare facility access across NYC boroughs to identify underserved areas and inform equitable resource allocation.

Key Questions

  • Which NYC boroughs have the greatest healthcare access gaps?
  • How do demographic factors correlate with facility availability?
  • What segmentation framework can guide resource allocation decisions?
